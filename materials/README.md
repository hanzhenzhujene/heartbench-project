# Materials

This directory collects the public-facing non-code artifacts for the project.

- `assets/`: homepage figures and visual assets
- `benchmark/`: benchmark files and relabel templates
- `research/`: configs, prompt sets, and internal reports
- `results/`: main experiment outputs and figures

If you only want the headline result, start with [results/staged_paper/heartbench_v2/main/](./results/staged_paper/heartbench_v2/main/).
