# Results

This directory contains the main experiment outputs used in the repository.

- `staged_paper/`: the paper-facing staged `J1 -> E -> J2` results
- `dual_logit_v2/`: the HeartBench-v2 dual-logit benchmark follow-on analyses
- `casewise_logit/`: corrected legacy benchmark results

If you only want the main current finding, start with [staged_paper/heartbench_v2/main/](./staged_paper/heartbench_v2/main/).
