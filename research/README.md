# Research Materials

This directory holds the internal research scaffolding that powers the public-facing results.

- `configs/`: experiment configurations and prompt registries
- `prompts/`: prompt templates and follow-up prompts
- `reports/`: internal memos, audits, benchmark notes, and revision packets

Most readers can start with the top-level [README.md](../README.md), the paper summary in [paper/](../paper), and the staged main results in [results/staged_paper/](../results/staged_paper).
